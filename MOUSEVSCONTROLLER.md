## Intro
Mouse vs controller has been argued for ages. In this document I'll prove with facts and logic that mouse is better.

## Mouse
The coordinates of a mouse are best described using cartesian coordinates where the physical quantity of both axes is distance: i.e. (d, d). Appropriate units would be (mm, mm)

## Controller
A controller uses joysticks. At first glance it might seem that those are best described with a 3D angle: i.e. (θ, θ).
This is not the case though. When a controller is held in a forward position, the pointer will keep moving.
With a calibrated mouse it is possible to determine the exact position of the pointer from the location of the mouse.
This is not the case with a joystick. To fully determine the position of the pointer, how long the joystick was held forward must be known.
Consoles could technically use the 3D angle to fully determine the pointer position, but this would be too inaccurate.
Instead they have the horizontal angle act as a multiplier for the time dimension. Thus we can derive the real coordinates: (θ, θt). Appropriate units would be (r, rs).

## Conclusion
To get to point A (e.g. a head) with a mouse, time is not a factor. You are only limited by how fast you can change the position.
To get to the same point with a controller, time is a factor. You are basically forced to wait.
